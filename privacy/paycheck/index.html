월급체크 개인정보처리방침

최종 수정일: 2026-04-29

본 문서는 Android 앱 월급체크의 개인정보 처리 방침입니다. 본 앱은 로그인 기능을 제공하지 않으며, 사용자가 직접 입력한 근무 기록과 설정 정보를 기기 내부에 저장합니다.

1. 앱 소개

월급체크는 근무 기록, 예상 급여 계산, 실제 입금 비교, 반복 입력, 알림, 백업 및 내보내기 기능을 제공하는 Android 앱입니다.

2. 수집 또는 저장되는 정보

본 앱은 회원가입이나 서버 계정을 사용하지 않으며, 운영자가 별도 서버로 사용자의 근무 기록을 직접 수집하지 않습니다.

  • 근무 기록
  • 시급 및 정산 시작일 설정
  • 실제 입금 비교 금액
  • 사용자 프리셋
  • 알림 설정
  • 언어 설정

위 정보는 기본적으로 사용자의 기기 내부 저장소에만 저장됩니다.

3. 백업 및 내보내기

본 앱은 사용자가 직접 선택한 경우에만 다음 파일을 생성할 수 있습니다.

  • JSON 백업 파일
  • CSV 파일
  • PDF 리포트

위 파일은 사용자가 직접 지정한 위치에 저장되며, 운영자에게 자동 전송되지 않습니다.

4. 광고 및 외부 서비스

본 앱은 광고 제공을 위해 Google AdMob SDK를 사용할 수 있습니다. 광고 제공 과정에서 Google 또는 관련 서비스 제공자가 광고 식별자, 기기 정보, 광고 상호작용 정보, 진단 정보를 처리할 수 있습니다.

관련 문서: Google 개인정보처리방침

5. 사용 권한

  • 인터넷 접근
  • 네트워크 상태 확인
  • 알림 표시
  • 기기 재부팅 후 알림 복원

본 앱은 위치, 연락처, 카메라, 마이크, 사진 라이브러리 등 민감한 권한을 요청하지 않습니다.

6. 알림

본 앱은 사용자가 활성화한 경우에만 근무 알림 또는 급여일 알림을 보낼 수 있습니다.

7. 아동 개인정보

본 앱은 아동을 대상으로 설계된 서비스가 아니며, 운영자가 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 데이터 삭제

사용자는 앱 삭제, 앱 데이터 초기화, 백업 파일 삭제를 통해 자신의 데이터를 제거할 수 있습니다.

9. 방침 변경

본 개인정보처리방침은 기능 변경, 법령 변경, 광고 SDK 구성 변경 등에 따라 수정될 수 있습니다.
